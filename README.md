VysokeSkoly/image-api
=====================

[![Build Status](https://travis-ci.org/vysokeskoly/image-api.svg?branch=master)](https://travis-ci.org/vysokeskoly/image-api)

Api for storing images

## Actions:
| action    | path      | method    |
| ---       | ---       | ---       |
| Homepage  | `/`       | ANY       |
| Auth      | `/auth`   | ANY       |
| Save      | `/image`  | POST      |
| Get       | `/image`  | GET       |
| Delete    | `/image`  | DELETE    |
